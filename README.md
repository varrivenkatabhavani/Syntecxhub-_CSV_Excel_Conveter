# CSV to Excel Converter
## Description
A CSV to Excel Converter is a Python-based application used to convert data from CSV files into Excel files efficiently using pandas and openpyxl libraries.
## Features
- Read CSV Files
- Convert CSV to Excel Format
- Handle Missing Values
- Automatic Excel File Generation
- Simple Data Management
- Input Validation
## Technologies Used
- Python
- Pandas
- OpenPyXL
- Command Line Interface (CLI)
## How to Run
1. Install Python
2. Install required libraries:
```bash
pip install pandas openpyxl
3.Save the file as csv_to_excel.py
4.Create a CSV file named sample.csv
5.Open terminal or command prompt
6.Run the program:
Bash
python csv_to_excel.py sample.csv output.xlsx
